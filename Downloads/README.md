# Downloads/

> Zona de aterrizaje temporal para archivos entrantes. Nada aquí se considera organizado de forma permanente.

---

## Uso previsto

- Esta es una bandeja de entrada, no una carpeta de almacenamiento. Los archivos aquí deben ser revisados y movidos a su ubicación correcta regularmente.

---

## Para Agentes IA

- No trates los archivos de esta carpeta como definitivos o autorizados. Confirma siempre con el usuario antes de usar un archivo de `Downloads/` en cualquier operación. Al ayudar a organizar, sugiere mover los archivos a su destino apropiado en lugar de trabajar con ellos aquí.
