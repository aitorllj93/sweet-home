# Audio/

> Recursos de audio organizados por categoría funcional

---

## Subcarpetas

| Carpeta | Descripción                             |     |
| ---------| -----------------------------------------| -----|
| `BGM/`  | Música de fondo — pistas en bucle       |     |
| `BGS/`  | Sonidos de fondo — ambientes            |     |
| `BGV/`  | Voz de fondo — canciones con letra      |     |
| `ME/`   | Efectos musicales — jingles, fanfarrias |     |
| `SE/`   | Efectos de sonido — sonidos de evento   |     |
| `VO/`   | Voz en off — voces de personajes        |     |

---

## Convenciones

- Formatos preferidos: `.ogg` (web/juego), `.wav` (master sin comprimir), `.mp3` (compatibilidad).

- Nombrado: `categoria_descripcion_variante.ext` — ej. `bgm_bosque_loop.ogg`

---

## Para Agentes IA

- Al añadir audio, coloca siempre los archivos en la subcategoría correcta. No dejes archivos directamente en `Audio/`.
