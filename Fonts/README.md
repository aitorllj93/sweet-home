## Para Agentes IA

- Cuando una tarea requiera una fuente específica, comprueba primero esta carpeta antes de sugerir descargas externas. Al referenciar fuentes, incluye siempre la ruta de subcarpeta, ej. `Fonts/FamiliaName-Regular.ttf`.
