Este es el directorio raíz de mi espacio de trabajo personal. Cada subcarpeta contiene su propio README con convenciones específicas.

---

## Estructura

```
~/
├── Audio/              # Assets de audio
├── Documents/          # Documentos personales y de trabajo
├── Downloads/          # Archivos entrantes temporales
├── Fonts/              # Tipografías
├── Graphics/           # Recursos gráficos
├── Notes/              # Notas rápidas y referencias
├── Projects/           # Trabajo organizado por ciclo de vida
├── Videos/             # Recursos de vídeo y media personal
└── README.md           # Este archivo
```

---

## Convenciones

- Los nombres de carpeta usan `PascalCase`. Los nombres de archivo usan `kebab-case` o `snake_case` según la herramienta que los consuma.
- Cada carpeta tiene su propio README.md describiendo su contenido, convenciones de nombrado y uso previsto.
