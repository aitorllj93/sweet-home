# Videos/

> Recursos de vídeo y media personal, organizados por tipo.

---

## Subcarpetas

| Carpeta     | Descripción                                                         |     |
| -------------| ---------------------------------------------------------------------| -----|
| `Movies/`   | Largometrajes, cinemáticas y referencias de vídeo de larga duración |     |
| `TV Shows/` | Series, episodios y contenido episódico                             |     |

---

## Uso previsto

- Esta carpeta tiene un doble propósito: biblioteca multimedia personal y material de referencia para proyectos creativos (cinematografía, gradación de color, referencias de edición).

---

## Convenciones

- Formatos preferidos: `.mp4`, `.mkv` (almacenamiento), `.mov` (masters de edición).

- Para series, usa estructura de subcarpetas: `TV Shows/NombreSerie/Temporada X/`

---

## Para Agentes IA

- Esta carpeta puede contener archivos de gran tamaño. Evita escanear o indexar su contenido completo a menos que se solicite explícitamente. Al referenciar contenido de vídeo para una tarea creativa, pide al usuario que especifique el archivo exacto en lugar de explorar la carpeta.
